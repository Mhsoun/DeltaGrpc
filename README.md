# Currency Delta gRPC API

## Overview

A gRPC API to calculate exchange rate deltas between a base currency and other currencies over a specified date range.

## Requirements

- [.NET 8.0 SDK](https://dotnet.microsoft.com/download/dotnet/8.0)
